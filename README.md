# homebrew-prumo

Tap do Homebrew para o [Prumo](https://github.com/wolffness/prumo).

```sh
brew tap wolffness/prumo
brew trust wolffness/prumo   # tap pessoal — confirmação única
brew install prumo
```

Instala o comando `prumo` (TUI + CLI todo.txt). Para o app completo do macOS
(Prumo.app, captura rápida ⌥] e barra de menu), clone o repositório e rode
`./scripts/package-macos.sh`.
